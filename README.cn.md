## 首次使用

### 第一步：获取 Square OpenAPI Key

1. 登录币安账号
2. 进入 **账号设置 → API 管理**
3. 创建新的 **Square OpenAPI Key**
4. ⚠️ **只勾选"发帖"权限**，不要勾选交易/提现权限
5. 将 Key 设置为环境变量：`SQUARE_OPENAPI_KEY=your_key_here`

### 第二步：安装 Skill

```bash
clawhub install binance-square-operator
```

或直接把本目录加入 OpenClaw 的 skills 路径。

### 第三步：开始使用

```
你: 分析广场爆款
Agent: 正在抓取最新热门帖子数据...（分析100条高浏览量帖子）
       ✅ 分析完成！发现以下规律：
       - 最佳开头：数字开头（"3个信号表明..."）效果最好
       - 最佳时段：UTC+8 09:00 和 20:00
       - 高频 Emoji：🚀 📈 💡 ⚠️ 🎯
       ...
       要用这个分析结果生成今天的帖子吗？
```

## 完整工作流

```
每天一次
    ↓
[分析] 抓取热门帖子 → 提炼爆款规律
    ↓
[生成] 结合当日市场热点 → 生成2个版本帖子
    ↓
[选择] 用户选择版本 A / B / 编辑
    ↓
[发布] 调用 Square OpenAPI 发布
    ↓
[等待 24 小时]
    ↓
[追踪] 抓取浏览/点赞/评论数据
    ↓
[优化] AI 分析效果 → 更新内容策略
    ↓
🔁 循环，持续提升内容质量
```

## 注意事项

- 每日发帖次数有上限（遇到错误码 220009 时当日已达上限）
- 目前仅支持纯文本，暂不支持图片
- 市场分析类帖子必须附加免责声明（Skill 会自动添加）
- 账号需完成 KYC 实名认证才能通过 OpenAPI 发帖

---

## 错误码参考

| 错误码 | 说明 | 解决方案 |
|--------|------|---------|
| 000000 | 成功 | — |
| 10005 | 未完成实名认证 | 完成 KYC |
| 20002 | 含敏感词 | 修改内容 |
| 20013 | 内容超长 | 缩短内容 |
| 220004 | API Key 已过期 | 重新生成 Key |
| 220009 | 今日发帖已达上限 | 明天再发 |
