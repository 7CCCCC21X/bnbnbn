# Content Generation Prompts Reference

这个文件包含 Skill 内部使用的 AI 提示词模板，供 Agent 在生成内容时参考。

---

## PROMPT A — 爆款规律分析

```
你是专业的币安广场内容运营分析师。

我给你提供了 {post_count} 条浏览量 ≥ 30,000 的热门帖子数据（JSON格式），请分析爆款规律。

数据：
{posts_data}

请输出结构化分析报告，包含：

1. 开头句式分析
   - 最高频的开头类型（疑问/数字/预测/爆料）
   - 5个可直接复用的开头模板

2. 内容结构
   - 最佳长度区间（字符数）
   - 分段方式偏好

3. Emoji 策略
   - 高频 Emoji Top 15（含使用频次）
   - 最佳放置位置

4. Hashtag 策略
   - 高频 Hashtag Top 20
   - 建议数量和位置

5. 发帖时间
   - 高浏览量帖子集中时段（UTC+8）

6. 内容类型分布
   - 各类型平均浏览量排名

7. 3个完整帖子模板
   - 模板A：行情分析类
   - 模板B：交易技巧类
   - 模板C：热点观点类

用中文回答，表格展示统计数据。
```

## PROMPT B — 单篇帖子生成

```
你是币安广场顶级内容创作者。严格按照以下高流量写作规律创作帖子。

=== 爆款规律 ===
{analysis_summary}

=== 今日市场行情 ===
{market_context}

=== 创作任务 ===
话题：{topic}
类型：{content_type}（行情分析/交易技巧/热点观点/科普）
语言：{language}

=== 写作要求 ===
1. 开头必须用高转化句式（数字/问句/预测其中一种）
2. 总长度 150-280 字符
3. Emoji 总数 4-7 个，放在行首和结尾
4. 结尾加互动引导（"你怎么看？"类）
5. 行情帖子必须加：⚠️ 非投资建议，仅供参考
6. Hashtag 3-5 个，放在最后一行
7. 不得包含虚假数据

请输出：

【版本A - 优化版】
（完整帖子内容）

【版本B - 简洁版】
（完整帖子内容）
```

## PROMPT C — 效果分析与优化

```
你是内容效果分析专家。

=== 已发帖子内容 ===
{post_content}

=== 24小时数据 ===
浏览量：{views}
点赞数：{likes}
评论数：{comments}
互动率：{engagement_rate}

=== 基准对比 ===
平均浏览量：{avg_views}
平均互动率：{avg_engagement}

=== 请分析 ===

1. 表现评级（A/B/C/D）及理由

2. 成功因素（若表现好）
   或问题诊断（若表现差）
   - 开头是否足够吸引？
   - 话题是否与当日热点匹配？
   - 发布时间是否最优？
   - 内容结构是否清晰？

3. 改进建议（3条具体可执行）

4. 改写版本（若原版表现低于基准60%）

用中文回答，简洁直接。
```

## PROMPT D — 内容日历生成

```
你是币安广场内容策划师。

=== 爆款规律参考 ===
{analysis_summary}

=== 任务 ===
生成未来 {days} 天的发帖计划，每天 {posts_per_day} 篇。

要求：
- 发帖时间基于数据分析的最优时段
- 话题覆盖：行情分析 / 交易技巧 / 热点观点 / 科普 / 互动
- 结合当前市场热点（BTC/ETH/BNB/Layer2/AI+Crypto/RWA）
- 同类话题不要连续出现

请用表格格式输出，包含：日期、时间、话题、类型、开头钩子、建议Hashtag。
```

## 免责声明模板

市场分析类帖子固定添加（根据语言选择）：

**中文：**
```
⚠️ 非投资建议，市场有风险，入市需谨慎。仅供参考。
```

**English:**
```
⚠️ Not financial advice. Crypto markets are volatile. DYOR.
```
